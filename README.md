# To-Do-List
A simple and lightweight command-line To-Do List application built with Python.

Add Tasks – Create new tasks in seconds.
View Tasks – Display all your current tasks in a clean, numbered list.
Delete Tasks – Remove completed or unwanted tasks by selecting their number.
Automatic Saving – Tasks are stored in a tasks.json file, ensuring they remain available even after closing the application.
Input Validation – Handles invalid inputs gracefully to prevent crashes and improve the user experience.
